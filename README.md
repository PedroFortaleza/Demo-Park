Demo Park API

A Demo Park API é uma aplicação back-end desenvolvida para gerenciamento e controle de estacionamentos. O projeto tem como objetivo facilitar o cadastro e gerenciamento de usuários, veículos, vagas e entradas/saídas de automóveis, oferecendo uma estrutura organizada e escalável para sistemas de estacionamento.

A API está sendo desenvolvida utilizando Java com Spring Boot, seguindo boas práticas de arquitetura em camadas, organização de código e desenvolvimento de APIs RESTful.

O projeto foi criado com foco em aprendizado e prática de desenvolvimento back-end, utilizando tecnologias modernas do ecossistema Java para construção de APIs robustas e organizadas.****
